# winterschool2026


Проєкт створений для людей, які мало розуміють сучасні технології, але щодня користуються інтернетом, смартфонами та месенджерами. У простій і зрозумілій формі він навчає, як розпізнавати скам, спам і фішингові атаки та як правильно діяти в небезпечних ситуаціях.

Матеріали подаються у вигляді реальних прикладів, інтерактивних сценаріїв і практичних підказок - без складних термінів і технічних деталей. Основний фокус проєкту - навчити мислити спокійно й критично, щоб не втрачати гроші, дані та доступ до своїх акаунтів.
